# PB-Incident-Assign-Group

## Description
Automatically assigns Microsoft Sentinel incidents to specific groups based on logic (severity, region, etc.)

## Trigger
Microsoft Sentinel Incident Trigger

## Files
- azuredeploy.json → ARM template
- parameters.json → Deployment parameters

## Deployment Steps
1. Go to Azure Portal
2. Deploy a custom template
3. Upload azuredeploy.json
4. Provide parameters.json values

## Use Case
- SOC automation
- Incident triage routing
- Region-based assignment
